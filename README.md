# Power BI Sales Dashboard

## Description
This project is an interactive **Sales Dashboard** created using **Power BI**.  
It visualizes sales data from an Excel file, including total sales, product performance, and state-wise distribution.

## Features
- **Key Metrics (Cards):** Orders count, Max Price, Total Price
- **Line Chart:** Monthly sales trend
- **Stacked Bar Chart:** Total sales per product
- **Donut Chart:** Sales distribution by state 
- **Map Visualization:** Total sales by state
- **Slicers (Filters):** Category

## Dataset
- Excel file: `SalesReport.xlsx`
- Columns included:  
  `Order ID, Product, Category, Quantity Ordered, Price Each, Total Price, Order Date, State`

## How to Use
1. Open the Power BI file: `SalesDashboard.pbix`
2. Connect to the Excel dataset
3. Interact with the dashboard using the filters and visualizations
